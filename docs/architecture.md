Folder structure:

```
├───docs - Documentation files
└───src - Source Code  
    ├───Core - Shared resources
    │   ├── Helpers
    │   ├── Models
    │   ├── Dtos
    |
    ├───Services - Backend layer with microservices
    │   └───MicroService - Project structure for a microservice
    │       ├───Api
    │       ├───Services
    │       │───Data
    │       └───Tests
```
